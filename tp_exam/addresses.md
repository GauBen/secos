# Virtual and physical addresses

## Physical addresses

| Address   | Content          |
| --------- | ---------------- |
| 0030 0000 | Multiboot header |
| 0030 0010 | Kernel stack     |
| 0030 2010 | Kernel           |
| 0060 0000 | Page directory   |
| 0060 1000 | Page tables      |
| 07ff ffff | Last address     |
